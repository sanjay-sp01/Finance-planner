# Finance-planner
The Personal Finance Goal Planner is a web-based application that helps users plan and manage their financial goals efficiently. It calculates the required monthly savings based on the target amount and time duration, enabling users to achieve their goals in a structured way.
Add financial goals (e.g., Phone, Laptop, Travel)

Set target amount and duration in months

Automatically calculates monthly savings required

View all added goals in a clear format

Delete goals when completed or not required

Persistent data storage using browser LocalStorage
